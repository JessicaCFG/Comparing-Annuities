## [1.0.0] - 2026-06-01

### Added
- **Core Annuity Comparison Engine** — Side-by-side comparison of Fixed, MYGA, and FIA products
- **Year-by-Year Projections** — Calculate interest, fees, surrender values, and income benefit base annually
- **FIA Modeling** — Support for participation rate, cap rate, and floor rate calculations
- **Income Rider Tracking** — Compounding benefit base roll-up with customizable rates
- **CSV Export** — Export all projections to Excel or Google Sheets compatible format
- **JSON Configuration** — Save and reuse client-specific scenarios
- **Surrender Charge Schedules** — Automatic lookups per year based on product data
- **CLI Interface** — User-friendly command-line arguments for customization
- **Example Scenarios** — Pre-built configuration files for quick starts
- **Documentation** — Comprehensive README with usage examples and config format
- **Contributing Guidelines** — Full CONTRIBUTING.md for community contributions
- **Python Package Setup** — requirements.txt for easy installation
